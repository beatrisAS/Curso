# Projeto Docentes - Página Web Interativa

Este projeto foi desenvolvido como parte de uma atividade acadêmica, visando a criação de uma página web interativa para apresentar informações sobre docentes de uma instituição. A página foi estruturada de forma a exibir um slideshow com fotos, nomes e uma breve descrição de cada docente, utilizando HTML, CSS e JavaScript. A implementação também inclui a organização das imagens em uma pasta específica e o uso de navegação manual para o slideshow.

## Funcionalidades

- **Seção de Docentes**: Apresenta um slideshow interativo com fotos e informações sobre os docentes.
- **Galeria de Imagens**: Armazena todas as fotos utilizadas no projeto, organizadas em uma pasta separada.
- **Slideshow Manual**: Os usuários podem navegar pelas imagens dos docentes utilizando botões de navegação.
- **Estilo Visual Consistente**: O projeto adota o padrão de cores `w3-cyan` para garantir uma boa visibilidade e estética visual.
- **Estrutura de Arquivos Organizada**: A estrutura do projeto foi organizada para facilitar o acesso e manutenção do código.

## Tecnologias Utilizadas

- **HTML**: Para estruturar a página e o conteúdo.
- **CSS**: Para estilizar e padronizar o design da página.
- **JavaScript**: Para criar a interação do slideshow e permitir a navegação manual entre as imagens.

## Estrutura do Projeto

O projeto contém os seguintes arquivos e pastas:

```
/Projeto-Docentes
│
├── /IMG                  # Pasta contendo as imagens dos docentes
│   ├── docente1.jpg
│   ├── docente2.jpg
│   └── docente3.jpg
│
├── index.html            # Arquivo HTML principal com o código da página
├── script.js             # Arquivo JavaScript para controle do slideshow
└── style.css             # Arquivo CSS para estilização da página
```

## Como Usar

1. **Clonar o repositório:**

   Para clonar o repositório para o seu computador, execute o comando abaixo:

   ```bash
   git clone https://github.com/beatrisAS/projeto-docentes.git
   ```

2. **Abrir o arquivo HTML:**

   Navegue até a pasta do projeto e abra o arquivo `index.html` no seu navegador para visualizar a página.
